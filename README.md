# 🚀 Application Suite

A collection of full-stack web applications developed as part of the CodeAlpha Internship Program. This repository demonstrates practical implementation of modern web development concepts using Django, Python, HTML, CSS, JavaScript, and database integration.

## 📌 Projects Included

### 1. Project Management System

A collaborative platform that helps teams organize projects and track progress.

**Features**

* User Authentication
* Project Creation & Management
* Task Assignment
* Task Status Tracking
* Team Collaboration
* Progress Monitoring

### 2. E-Commerce Store

An online shopping platform with product browsing and management features.

**Features**

* Product Catalog
* Shopping Cart
* User Accounts
* Order Management
* Responsive Design

### 3. Social Media Platform

A social networking application for user interaction and content sharing.

**Features**

* User Profiles
* Post Creation
* Like & Comment System
* Feed Management
* Authentication

### 4. Expense/Activity Tracker

A tracking application for managing daily activities and records.

**Features**

* Record Management
* Data Visualization
* User-Friendly Interface
* Progress Tracking

---

## 🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Python, Django
* Database: SQLite

## 🚀 Getting Started

1. Clone the repository

git clone https://github.com/sanyukt63/CodeAlpha_Application-Suite.git

2. Navigate to the project folder

cd CodeAlpha_Application-Suite

3. Install dependencies

pip install -r requirements.txt

4. Run the server

python manage.py runserver
